Thai Speaking 1｜Day 12 學習指南

主題：曾經做過、做過幾次、第一次/最後一次、早中晚時間段、昨晚、幾點鐘、出發/抵達、開門/關門

Textbook-based Supplement / Local usage Review Common mistake

Lesson Overview

Textbook-based Day 12 的核心是兩條線：第一條是「經驗」，例如你曾經去過哪裡、吃過什麼、做過幾次、第一次或最後一次是什麼時候。第二條是「時間」，包含早上、上午、中午、下午、傍晚、晚上、深夜，以及泰式幾點鐘說法。

Review： Day 11 已經學過「多久、星期、日期、生日」。Day 12 會把時間變得更口語、更真實：你會開始聽到 กี่โมง / gìi moong / what time、กี่ทุ่ม / gìi thûm / what hour at night、ตอนเช้า / dtɔɔn cháao / morning。
Bangkok Practical Focus： 這課最實用的場景是：問朋友去過哪裡、問餐廳幾點開/關、問昨晚幾點吃飯、問 Grab 或朋友幾點出發/到達、說自己第一次去某地、最後一次去某店是什麼時候。

Sentence Structures

Textbook-based
คุณเคย + V. + Object + ไหมคะ
khun khəəi + V. + Object + mái khá

QuestionPositiveNegative

เคยไปแล้วค่ะ

ไม่เคยไปค่ะ

Textbook-based

3. First time / nth time / last time + when?

Textbook-based
คุณ + V. + Object + ครั้งแรก / ครั้งที่... / ครั้งสุดท้าย + เมื่อไรคะ
khun + V. + Object + khráng rɛ̂ɛk / khráng thîi... / khráng sùt-tháai + mʉ̂a-rài khá
When did you do it for the first / nth / last time?
QuestionAnswer
คุณไปเชียงใหม่ครั้งแรกเมื่อไรคะ
khun bpai Chiang Mai khráng rɛ̂ɛk mʉ̂a-rài khá

ไปครั้งแรกเมื่อสองปีก่อนค่ะ

I went for the first time two years ago.
คุณไปครั้งสุดท้ายเมื่อไรคะ
khun bpai khráng sùt-tháai mʉ̂a-rài khá
When did you go last time?
ไปครั้งสุดท้ายเมื่อเดือนก่อนค่ะ
bpai khráng sùt-tháai mʉ̂a dʉan gɔ̀ɔn khâ
I went last time one month ago.

 加 นะ 會比較柔和，像再確認。

 你站在店門口或指著店時，可以省略ร้าน。

Common Mistakes & Quick Review

Mistake Why It’s Wrong / Risky Better Version
เคยไม่ไปค่ะ
khəəi mâi bpai khâ
wrong: ever not go
 「從來沒有」要用 ไม่เคย，順序不能反。
ไม่เคยไปค่ะ
mâi khəəi bpai khâ
I have never been.
เคยไปหนึ่งครั้งแล้ว
khəəi bpai nùeng khráng lɛ́ɛw
I have been once already.
 不是錯，但口語如果想說「只去一次」，更自然用 ครั้งเดียว。
เคยไปครั้งเดียวค่ะ
khəəi bpai khráng diao khâ
I have been once only.
ครั้งแรก = ครั้งสุดท้าย
khráng rɛ̂ɛk = khráng sùt-tháai
first time = last time
 兩個完全不同：第一次 vs 最後一次。
ครั้งแรก
khráng rɛ̂ɛk
first time
ครั้งสุดท้าย
khráng sùt-tháai
last time
เปิด ≠ ปิด
bpə̀ət ≠ bpìt
open is not close
 發音很近，意思相反。開是 เปิด，關是 ปิด。
ร้านเปิดกี่โมงคะ
ráan bpə̀ət gìi moong khá
What time does the shop open?
ร้านปิดกี่โมงคะ
ráan bpìt gìi moong khá
What time does the shop close?
หนึ่งทุ่ม = 1 p.m.
nùeng thûm
wrong: 1 p.m.
 ทุ่ม 是晚上系統，หนึ่งทุ่ม 是 7 p.m.。
หนึ่งทุ่ม
nùeng thûm
7 p.m.
ตีหนึ่ง = 1 p.m.
dtii nùeng
wrong: 1 p.m.
 ตี 是凌晨系統，ตีหนึ่ง 是 1 a.m.。
ตีหนึ่ง
dtii nùeng
1 a.m.
本課最重要記憶點：

1. เคย / khəəi / have ever 問經驗；否定是 ไม่เคย / mâi khəəi / have never。

2. ครั้งแรก / khráng rɛ̂ɛk / first time；ครั้งสุดท้าย / khráng sùt-tháai / last time。

3. กี่โมง / gìi moong / what time 是最通用；晚上要慢慢熟悉 ทุ่ม / thûm / evening hour 和 ตี / dtii / early morning hour。
